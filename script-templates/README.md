# Script Templates

This folder contains script templates designed specifically for use within
**Workstation**. These templates provide starting points for common tasks
within the Strategy One environment.

## Important Note

The templates in this folder are built to work within the Workstation context.
If you intend to use these templates in a standalone environment (outside
Workstation), you will need to:

- Rework them to accommodate standalone execution requirements
- Update any Workstation-specific dependencies or configurations
- Adjust authentication and connection handling as needed

Some templates may not be functional outside of Workstation without
significant modifications.
